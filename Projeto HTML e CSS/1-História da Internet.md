A internet nasceu no período da guerra fria, a união soviética lançou o sputinik, nisso o Dwight Elisnhower ele fundou uma agência para estudo de tecnologia (DARPA) estudo de tecnologia para guerra, ela cria tecnologia e armas para guerra. Em 1969 foi criado uma tecnologia pela DARPA para resolve um problema para EUA de problema de backup de dados, com isso ele criaram uma pequena rede chamado APANET para fazer um backup gigante entre bases militares e para manter a comunicação entre eles, nisso ele criam um protocolo NCP pois ela fala uma linguagem única por conta que os computadores das bases eram diferentes um do outro e tinham línguas diferentes. Em 1972 rede foi crescendo e foi inviável utilizar o protocolo NCP, com isso dois pesquisadores Bob Kahn criou o protocolo TCP que veio com o objetivo de quebrar o problema de ter que parar a rede para fazer a transmissão, mas esse protocolo tinha grande problema pela rede ser grande demais e ele tinha problema de identificação de redes, com isso surgiu outro pesquisador Vint Cerf que criou um protocolo de identificação de máquinas (IP), juntando o trabalho dos dois pesquisadores foi criado o protocolo TCP/IP que utilizamos até nos tempo de hoje, só q o protocolo já está numa versão mais evoluída, hoje eles está na sua versão 6. Em 1977 teve a primeira comunicação via satélite com Londres, com isso a arpanet se tornou uma rede muito grande pois começou a se conectar com continentes, então os EUA decidiu dividir as redes: MILNET(militar), NSFNET(científica) e COMERCIAL(público). Todas essas rede tinham que se conectar, menos a rede MILNET que era uma rede isolada. Com isso podemos dizer q a internet surgiu como uma forma de defender bases militares.
